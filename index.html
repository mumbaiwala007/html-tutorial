What have I learnd so far in HTML?

HTML stands for Hyper Text Markup Language and is used to create web pages.


HTML uses what are called "tags" or "HTML elements" to render web pages.
  • Header and paragraph elements (contain text contents)
  • Void elements (no content) new lines and horizontal rule
  • Image and anchor (hyper link) elements
  • Ordered and unordered lists

Basic tags

The header tags

There are six header tags starting from h1 to h6. There should be only one h1 in a document which is equivalent to title of a book for example. This is also important from accessability point of view for the folks who use screen readers. See examples below:

This is header 1 and is repeated in this web page, sorry!

This is header 3.

This is header 4.

This is header 5.
This is header 6.
This is header 7. <== There is no such thing and this is rendered as "normal text".

This is how a typical header tag looks:
<h1>Your text</h1>
<h2>Your text for an h2 tag</h2>
<h3>Your text rendered as an h3 tag</h3>
And so on...
